## 3.0常见命令
/join <群组/频道链接或 ID>：加入指定的群组或频道。
/leave <群组/频道链接或 ID>：退出指定的群组或频道。
/add_listen <群组/频道链接或 ID> <目标机器人用户名或 ID>：将指定的群组或频道添加到监听列表，并指定转发的目标机器人。例如：/add_listen @channel_A @bot_b_username
/remove_listen <群组/频道链接或 ID>：将指定的群组或频道从监听列表中移除。
/list_listen：显示当前所有正在监听的映射。

## 概述

本项目使用python基于Telegram Bot API的**Telegram频道偷鸡机器人**，可以同时向多个频道/群组发布相同内容，实现一站发布、全局覆盖的效果。
#### 使用方法为:设置好自己的源频道或者直接设置为自己的id,把别人频道的视频/资源转发给机器人,让机器人可以快速批量的复制高质量的频道内容

## 核心优势

### 规模化运营
- **批量管理**：统一管理多个Telegram账号/机器人
- **同步发布**：一次操作，多频道同时更新
- **效率提升**：大幅减少重复操作时间,只需要从别人的频道转发给机器人,实现一次转发,多次复制出来一模一样的频道

### 搜索引擎优化
- **索引抢占**：通过频繁更新占据全局搜索
- **收录加速**：多频道同时活跃提升搜索引擎抓取频率
- **自然流量**：提高在Telegram内置搜索中的曝光率

### 配置管理
- **模块化设计**：每个配置文件独立管理一个机器人token
- **灵活扩展**：轻松添加新的发布渠道
- **隔离运行**：单个配置文件故障不影响其他机器人

## 部署方案
